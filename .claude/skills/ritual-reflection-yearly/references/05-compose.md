## Confirmation Preview

Present all proposed changes for user approval:

1. **Archive destination**: Periodic/Yearly/YYYY.md
2. **Year narrative**: The one-paragraph summary
3. **Semantic updates**: List each file, section, and content preview
4. **Year.md replacement**: Archived placeholder

Options:
- **Proceed all** - archive and apply all updates
- **Archive only** - skip semantic updates
- **Review each** - approve individually
- **Cancel** - exit without changes (completed review preserved in Year.md)
