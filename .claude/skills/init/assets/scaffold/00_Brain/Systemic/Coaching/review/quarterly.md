## Patterns

### Outcome Assessment

Review quarterly outcomes:

- Which were achieved?
- Which fell short and why?
- What would you do differently?

### Coaching Synthesis

Synthesize coaching patterns:

- Which patterns to watch emerged most?
- Which coaching questions served best?
- What new patterns were discovered?

### Theme Reflection

Reflect on quarterly theme:

- How well did the theme guide decisions?
- What would you name this quarter in retrospect?
- What theme calls for next quarter?

## Questions

- "What are you most proud of this quarter?"
- "What surprised you?"
- "What would you tell yourself at quarter start?"
