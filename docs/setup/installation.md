## Fresh install

1. **Clone the engine**

   ```bash
   git clone https://github.com/bugroger/2bd 
   cd 2bd 
   ```
2. **Create your vault**

   ```bash
   claude skill run init
   ```

   This guides you through initalizing the system.

3. **Run your first ritual**

   ```bash
   claude skill run ritual-planning-daily
   ```
